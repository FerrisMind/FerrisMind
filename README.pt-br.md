## 🎨 Na Junção dos Mundos

Meus projetos são uma tentativa de encontrar equilíbrio entre capacidades técnicas e expressão visual. O design de interface do usuário é o lugar onde a tecnologia encontra a arte, onde algo verdadeiramente belo e significativo pode ser criado.

> ### 🏔️ **Picos Montanhosos — Fundação Confiável**
>
> Cada solução deve permanecer firme como rocha montanhosa. Busco criar aplicações que funcionem de forma estável e resistam ao teste do tempo.

> ### 🌌 **Azul Cósmico — Profundidade da Interface**
>
> Me inspiro na natureza multicamada e profundidade das obras de Roerich. Quero que as interfaces do usuário sejam igualmente pensadas — intuitivas na superfície, mas com atenção a cada detalhe.

## 🌋 Oficina de Projetos

### 🦀 [Laboratório Oliva (Oxide Lab)](https://github.com/FerrisMind/Oxide-Lab)

Aplicação desktop moderna em **Rust + Tauri v2 + Svelte 5**, criada para comunicação com modelos de IA que funcionam **completamente localmente** em seu computador. Sem assinaturas, sem dados na internet — apenas você e seu **assistente de IA pessoal**.

**Recursos:** 🔒 Privacidade completa • 🎨 UI/UX moderna • 🛡️ Sem assinaturas

---

### 🤖 [Inspector-GGUF](https://github.com/FerrisMind/inspector-gguf)

Ferramenta poderosa para análise e visualização de metadados de **arquivos GGUF** usados em aprendizado de máquina. Usa a biblioteca **Candle** para trabalho eficiente com modelos.

**Recursos:** 🔍 Análise profunda • 🖼️ Interface gráfica • ⚙️ Modo CLI

---

### 🌐 Outras Criações

Novas ferramentas para trabalhar com **aprendizado de máquina** e **tecnologias web de nova geração** estão em desenvolvimento. Cada projeto é um passo para realizar o sonho de coexistência harmoniosa entre humanos e tecnologia.
